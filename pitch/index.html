Terms

Virtualization

A level of virtual abstraction from hardware or the physical.

Virtual Environment

An isolated instance create by virtualization which allows for the instance to not affect operating instance running parallel with it.

Computing Paradigm

Refers to the current computing model being used in a consistent pattern.

Virtual Machine Monitor

A piece of firmware which manages, maps and allocates host resources of virtual environments. (aka a Hypervisor)

Hypervisor Server

A virtual machine manager, which can use the host computing resources to service client devices.

Terms

Virtual Desktop Agent

A desktop client which uses a virtual resource infrastructure to access the computing resources of a host virtual machine.

Remote Desktop Client

A client program which allows a user to connect to a computer in another location, see that computer's desktop and interact with it as if it were local.

Virtual Dedicated Graphics Acceleration

A single virtual machine mapped to a physical host GPU which provides high-end, hardware-accelerated graphics through a network to a discrete client device.

Virtual Shared Graphics Acceleration

Many virtual machines mapped to a physical host GPU which provides high-end, hardware-accelerated graphics through a network to a discrete client device.

Asynchronous Local Area Network System

A centralized server system which can virtually server multiple light weight, thin-client devices through a local area network with the computer resources necessary to function as full range computers.
